Bibliometri

Cite.py

This tool takes Web of Science data and creates an easy to read table that can be sorted on citations and more. It also creates keyword- and author citation networks in handy .gexf files that can be opened in Gephi.

Launch Citepy

Altmetric analyzer

This tool takes Web of Science, Scopus or a list of DOI numbers and then calls the Altmetric API to retrieve Altmetric score, citations from social media and a range of other sources. Exports an xlsx and csv files for further analysis.

Launch Altmetric analyzer
